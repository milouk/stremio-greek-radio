## stremio-greek-radio
Greek Radio Stremio addon

## Installation
### Manual 
Just copy and paste the [Manifest](https://stremio.duckdns.org/manifest.json) URL on "Search Addon" search box on stremio.

### Community Addons
Seach "Greek Radio" on the stremio community addons.
